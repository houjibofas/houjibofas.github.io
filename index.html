2.1、git在本地操作：
2.1.1、git命令框内可使用linux命令。
2.1.2、git init：初始化一个git仓库。
2.1.3、git status：查看暂存区有哪些文件，及存在的状态。
2.1.4、git add：添加一个文件到暂存区，可用git status查看暂存区文件存在的状态。
2.1.4、git commit -m '描述'：提交暂存区的（增删改）文件到git仓库，可用git status查看暂存区文件存在的状态。
2.1.5、git rm：从git仓库中删除一个文件。
2.1.6、git config --list：查看配置信息。

2.2、将本地仓库提交到远程仓库（需要git push命令）：
2.2.1、git clone 远程仓库地址：复制远程仓库到本地git仓库。
2.2.2、使用本地git操作命令之后，使用git push命令提交到远程仓库。（权限是远程仓库的账号密码）
